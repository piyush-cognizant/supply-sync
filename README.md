# Project Setup Guide

## Clone the Repository

```bash
git clone https://github.com/piyush-cognizant/spinkart.git
cd spinkart
```

## Install Dependencies

```bash
npm install
```

## Run the Project

```bash
npm run dev
```

## Additional Commands

- **Build**: `npm run build`
- **Test**: `npm test`

---

**Note**: Replace `<repository-url>` and `<project-name>` with your actual repository details.
